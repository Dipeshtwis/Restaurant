## Getting Started <a name = "gs"></a>

To get a local copy of the repository please run the following commands on your terminal:

```
$ cd <folder>
```

~~~bash
$ git clone https://github.com/Dipeshtwis/Restaurant.git
$ cd Restaurant

~~~

## For Webpack

~~~bash

$ npm run build

$ npm run start

~~~
